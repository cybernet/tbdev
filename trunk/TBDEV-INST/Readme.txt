Quick notes for installing this Pre-Coded Tbdev Release By Bigjoos
Edit 401, 402, 403, 404.html files
Edit all index.html files and set your url in the .htaccess files in /bitbucket - /bitbucket/images to disable hot-linking
No CHMOD required for Windows Installs
On a linux install Set CHMOD777 for folders and files in the following -->... /bitbucket /bitbucket/thumbs /cache /cache/index cache/topten /dox /forumattaches /torrents /logs /dir_list /settings and the two nested files  ... Also include/cache and the three files it contains.. imdb/cache and imdb/images
The /include/secrets.php must be writable (CHMOD 666)
Do not change the Default username -> Admin login on install - its part of a staff protection function - before promoting any staff on site ensure they are added using the onsite staff manager first.
Strongly Advised to run this code on a server only !!
//////////////////////////////////////////////////////////////////////////////

2. Follow on screen steps
3. Once installed make sure to chmod or delete the install directory
